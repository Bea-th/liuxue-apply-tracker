# 留学申请进度追踪 · 本地版

这是一个只在本机运行的留学申请进度追踪网站。前端来自 v5 HTML，数据不再保存在浏览器 `localStorage`，而是自动写入本项目里的 `data/data.json`。

## 准备（只需一次）

1. 安装 **Node.js（LTS 版）**：到 https://nodejs.org 下载安装。安装后重开终端。
2. 验证安装：在终端输入 `node -v`，能看到版本号即可。

## 安装依赖（只需一次）

打开终端，进入本项目文件夹：

```powershell
cd 具体地址
npm install
```

## 启动（每次使用）

```powershell
npm start
```

启动后终端会显示网址，并尽量自动打开浏览器；若没有自动打开，手动访问：

```text
http://localhost:5173
```

这就是你的本地网站。打开后即可使用，所有改动会自动保存到 `data/data.json`。

## 日常使用

- **首页**：国家/地区网格，每个卡片显示该国学校数、专业数和状态分布；点卡片进入该国。
- **国家页**：学校列表，点学校卡片就地展开，在下方填写或查看该校的专业，并对每个专业设置状态。
- **看板 / 清单**：右上切换视图；顶部流水线可按状态筛选；搜索框可搜国家、学校或专业。
- **增删**：可添加、编辑、删除国家、学校、专业。
- 左上角状态提示会显示“已保存 · 本地文件”或“保存中…”。

## 备份你的数据

- 推荐方式：直接复制 `data/data.json` 到别处，例如 U 盘、网盘或另一个文件夹。
- 也可以在网页里点“导出备份”，下载一个 JSON 文件保存好。
- 恢复时，把备份的 `data.json` 覆盖回 `data/`（先停止服务再覆盖），或在网页里点“导入备份”。

## 停止

在运行 `npm start` 的终端里按 **Ctrl + C**。

下次使用时再次运行 `npm start`，数据仍然在 `data/data.json`。

## 换一台电脑使用

把整个 `test1` 文件夹复制过去，包含 `data/`。新电脑装好 Node.js 后，在该文件夹里运行：

```powershell
npm install
npm start
```

## 常见问题

- **`node` 不是命令 / 找不到**：没装 Node.js，或安装后没有重开终端。
- **端口被占用**：换端口启动。
  - Windows PowerShell：`$env:PORT=5174; npm start`
  - Windows CMD：`set PORT=5174 && npm start`
  - macOS / Linux：`PORT=5174 npm start`，然后访问 `http://localhost:5174`
- **字体没加载 / 样式略有差异**：页面用了在线字体，断网时会自动回退到系统字体，不影响使用。
- **提示“保存失败”**：确认运行 `npm start` 的终端还开着，服务没有被关，并确认 `data/` 文件夹可写。
- **隐私**：服务器只监听 `127.0.0.1`，只有本机能访问，不会上线，不向任何服务器上传数据。
